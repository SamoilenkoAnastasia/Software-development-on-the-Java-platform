# Лабораторна робота №2: Аналіз хтонічних істот (Java Stream API та Gatherer)

### Опис проекту
Цей проект реалізує завдання з обробки та аналізу даних, використовуючи сучасний **Java Stream API** (з JDK 22+). Доменна область — **Хтонічні істоти** (міфічна нечисть).

### Виконані завдання
1.  **Генерація:** Створено нескінченний потік об'єктів `MythicCreature` з реалістичними випадковими параметрами.
2.  **Gatherer (Custom Intermediate Operator):** Реалізовано користувацький оператор потоку (`Gatherer`) для генерації списку з 500 елементів, пропускаючи перші $N$ елементів, що відповідають певному критерію (за видом істоти).
3.  **Фільтрація та Групування:** Список відфільтровано за віком першої згадки (Поле Б) та згруповано за видом (Поле В).
4.  **Користувацький Collector:** Створено `AttackStatsCollector` для обчислення мінімального, максимального, середнього значення та **стандартного відхилення** сили атаки (Поле Г).
5.  **Аналіз Викидів (Outlier Analysis):** Проведено аналіз **міжквартильного розмаху (IQR)** для ідентифікації викидів у силі атаки.

### Технології та вимоги

* **Мова:** Java 22+ (Потрібна для підтримки `java.util.stream.Gatherer`).
* **Система збірки:** Apache Maven 3.x
* **Main-клас:** `com.kpi.App`
